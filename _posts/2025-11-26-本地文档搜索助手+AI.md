## **02 操作步骤**

### 1. 本地文档搜索助手数据表

本地文档搜索助手的数据库用的是 **PostgreSQL**， 主要的数据表 **documents** 的主要字段如下

| 字段名                 | 描述                                                         |
| ---------------------- | ------------------------------------------------------------ |
| **id**                 | 自增主键                                                     |
| **file_name**          | 文件名                                                       |
| **file_type**          | 文件类型扩展名                                               |
| **file_size**          | 文件大小，单位：字节                                         |
| **file_created_at**    | 文件创建时间                                                 |
| **file_modified_time** | 文件最后修改时间                                             |
| **file_path**          | 文件绝对路径（跨平台存储统一为/分隔符）                      |
| **source_url**         | 文章链接，对于从网络保存的文章的链接，如微信公众号文章的链接 |
| **markdown_content**   | 转换后的 markdown 内容                                       |



### 2. 在 VS Code 安装 PostgreSQL 扩展

##### 2️⃣ 问题2

请总结一下"主要相关文章/演讲实录"的文章

![image-20251128173301208](2025-11-26-本地文档搜索助手+AI.assets/image-20251128173301208.png)



```markdown
以下是对上述“主要相关文章/演讲实录”的总结。

需要特别说明的是，经过对内容的详细阅读，其中 ID 2564 和 ID 2625 是王坚院士本人的演讲实录；而 ID 1937、ID 1938 和 ID 1951 则是阿里技术专家（如吴翰清/道哥）基于王坚博士的理念进行的深度解读或相关思考。

一、 王坚院士亲述演讲
1. 王坚院士在2024年外滩大会上的AI演讲 (ID: 2564)
核心主题： AI、AI+ 与 AI基础设施

AI的历史观： 王坚认为AI“有很长的过去，但只有很短的历史”。真正的现代AI历史始于2017年 Transformer 模型的提出，它改变了AI的格局。
关于“AI+”： “AI+”不应只是简单地将AI叠加到某个行业上，而是一种机制的创新。他以 OpenAI 为例，认为其非营利与商业结合的复杂机制是创新的关键。
AI基础设施： 这是技术渗透的终极形式。当数据、模型和算力的规模都发生巨大变化（超过1000倍）时，必须引入“AI基础设施”的概念。
数据的重新定义： 数据不仅是模型或计算的附属品，而是基础设施的核心组成部分。只有当数据、模型、算力结合成为完整的基础设施时，才会产生激动人心的创新。

2. 王坚院士最新演讲：计算科学时代已经到来 (ID: 2625)
核心主题： 计算科学（Computational Science）、AI与研究范式变革

计算科学时代： 我们已进入“计算科学”时代，其特征是计算密集型、数据驱动和基于模型的协同。
诺贝尔奖的启示： 2024年诺贝尔物理学奖和化学奖授予AI相关研究，标志着AI已成为科学研究的重要工具，改变了研究范式（从假设驱动转向数据驱动）。
规模（Scale）的重要性： 规模不仅仅是数量的增加，量变会引起质变（如 AlphaFold 的成功）。
AI与人的关系： AI不是要取代人类，而是通过计算能力扩展人类的创造力，帮助人类探索未知（如哈勃望远镜拍摄“空白”区域发现新事物）。

二、 他人解读与延伸思考
1. 吴翰清（道哥）关于“计算”的分享 (ID: 1937, 1938)
文章背景： 这两篇文章是吴翰清在2021年全球架构师峰会及2050大会上的分享，他回顾了在阿里云早期王坚博士提出的“计算作为公共服务”的愿景。
核心观点：
计算的本质： 目前的计算主要是在模拟现实世界（如淘宝模拟购物），而未来的计算将走向创造（如构建元宇宙、新世界）。
阿里云大脑构想： 提出在算力基础设施之上，未来需要一层“知识库”和计算引擎，能够进行逻辑推理和知识关联。
对王坚理念的感悟： 承认自己花了十年才真正理解王坚博士当年所说的“计算”不仅仅是计算机，而是一种通用的公共服务能力。
2. 我“重新”理解的云计算 (ID: 1951)
文章背景： 作者蔺瑞军受王坚院士著作《在线》及吴翰清文章的启发，重新梳理了对云计算的理解。
核心观点：
算力与计算： 区分了算力（物理演进、神龙架构、盘古存储等）与计算（弹性计算、大数据、智能计算等应用场景）。
在线（Online）： 引用王坚《在线》中的观点，强调大数据的本质在于“在线”，即输入和输出的双向在线。
基础设施的特征： 云计算作为基础设施，关键在于信任、低成本和自服务能力，就像电力一样普及。
```
